# 💧 Приключения Кап-Кап

Лендинг для детской книги **«Приключения Кап-Кап»** — добрая история о круговороте воды в природе.

---

## 📖 О книге

**«Приключения Кап-Кап»** — детская книга для малышей от 3 лет.

Вместе с капелькой Кап-Кап ребёнок узнает, как вода путешествует по кругу: становится дождём, снегом, росой и паром — и снова возвращается на небо.

**В книге есть:**

- 📖 Добрая сказка с яркими разворотами
- 🎨 Акварельные иллюстрации на каждой странице
- 🧪 Простые опыты с водой для детей и взрослых
- 🔬 Блок «Знаешь ли ты?» — наука простыми словами

**Автор:** Екатерина Макарова  
**Возраст:** от 3 лет  
**Формат:** электронная + печатная  
**Год:** 2026

---

## 🌐 Лендинг

Лендинг собран в двух файлах `index.html` + `style.css` — никаких зависимостей, открывается в любом браузере.

**Что есть на странице:**

- Обложка книги и слоган «Вода, которая всегда рядом»
- Блок «О книге» с текстом для родителей
- Отрывок из главы 1 «Рождение Кап-Кап»
- Раздел «Загляни в книгу» (лето, зима, круговорот, особенности)
- Блок автора — Екатерина Макарова
- Раздел «Для кого» (малыши / родители / в подарок)
- Кнопки покупки (заменить на реальные ссылки при выходе)
- Полностью адаптивный дизайн (телефон + компьютер)
