# 💧 Путешествие Кап-Кап

Лендинг для детской книги **«Путешествие Кап-Кап»** — добрая история о круговороте воды в природе.

---

## 📖 О книге

**«Путешествие Кап-Кап»** — детская книга для малышей от 3 лет.

Вместе с капелькой Кап-Кап ребёнок узнает, как вода путешествует по кругу: становится дождём, снегом, росой и паром — и снова возвращается на небо.

**В книге есть:**

- 🎵 Ритмичные стихи с хлопками и звуками
- 🎨 Акварельные иллюстрации на каждой странице
- 🧪 Простые опыты с водой для детей и взрослых
- 🔬 Блок «Знаешь ли ты?» — наука простыми словами

**Автор:** Екатерина Макарова  
**Возраст:** от 3 лет  
**Формат:** электронная + печатная  
**Год:** 2026

---

## 🌐 Лендинг

Лендинг собран в двух файлах `index.html` + `style.css` — никаких зависимостей, открывается в любом браузере.

**Что есть на странице:**

- Анимированная Кап-Кап — падает как дождик на обложке книги
- Дождь в герой-секции, снег в разделе с отрывком
- Отрывок стихов из первой главы
- Раздел с особенностями книги (стихи, иллюстрации, опыты, наука)
- Блок автора — Екатерина Макарова
- Раздел «Для кого» (малыши / родители / в подарок)
- Кнопки покупки (заменить на реальные ссылки при выходе)
- Рябь на воде в секции покупки
- Кастомный курсор в виде капельки 💧
- Полностью адаптивный дизайн (телефон + компьютер)

---

## 🚀 Публикация на GitHub Pages

1. Убедись, что файлы `index.html` и `style.css` находятся в корне репозитория
2. Перейди в **Settings → Pages**
3. Выбери ветку **main** → папку **/ (root)**
4. Нажми **Save**
5. Сайт появится по адресу: `https://katerinamakart.github.io/Book-litres-start`
